# Human Resource Management System (HRMS)

## Description
The Human Resource Management System (HRMS) is a web application designed to manage employee records and attendance tracking efficiently. It allows users to add new employees, mark their attendance, and generate department reports.

## Technologies Used
- **Backend**: Flask
- **Database**: SQLite
- **Frontend**: HTML, CSS, JavaScript

## Features
- Add new employees with details like name, designation, department, and date of joining.
- Mark attendance for employees with options for "Present" or "Absent".
- Search for employees by name.
- View attendance records for individual employees.
- Generate a department report showing the number of employees in each department.
